Nanoparticle Builder

Welcome to the interactive nanoparticle builder. Please note, that this tool is still under development.

The tool uses the program DISCUS to generate the nanoparticle structures and calculate the corresponding atomic pair distribution function and powder diffraction pattern. The interactive structure viewing is using the program JMOL. Structure and calculated functions can be downloaded from the results page. Online plotting of the powder pattern and pair distribution function use the Ptplot package. More builders for more complex nanoparticles are coming soon ..

For questions and comments, contact one of the team member listed on the left. Select a builder from the list on the left and get started ..
